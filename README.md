#  Number Guessing Game (C++)

## Description

Ce projet est un petit jeu en console développé en C++ durant ma première année d’études.

L’ordinateur choisit un nombre aléatoire entre **0 et 100** et le joueur doit essayer de deviner ce nombre.

Le joueur reçoit des indications après chaque tentative :

* **C'est plus !**
* **C'est moins !**

Le jeu se termine lorsque le joueur trouve le nombre ou lorsque le nombre maximum d'essais est atteint.

---

##  Niveaux de difficulté

Le jeu propose trois niveaux :

* **Facile** : 30 essais
* **Moyen** : 20 essais
* **Difficile** : 10 essais

---

##  Concepts utilisés

Ce projet utilise plusieurs concepts de base du langage C++ :

* Boucles (`do...while`)
* Conditions (`if / else`)
* Structure de contrôle (`switch`)
* Génération de nombres aléatoires (`rand`)
* Entrée / sortie (`cin` / `cout`)

---

## ⚙️ Compilation

Pour compiler le programme :

```bash
g++ main.cpp -o game
```

---

##  Exécution

```bash
./game
```
